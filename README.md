# epublib-android
[epublib](https://github.com/psiegman/epublib) port for Android

## Gradle
```gradle
repositories {
        maven {
            url  "http://dl.bintray.com/dasar/maven"
        }
    }
```

```gradle
compile 'nl.siegmann.epublib:epub-core:1.0@aar'
```

## Requirements

Android 2.3 (Gingerbread) and later.
